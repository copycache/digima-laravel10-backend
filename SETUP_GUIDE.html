Project Setup Guide

This guide describes how to set up the Domus Naturae project locally. It covers both the Laravel 10 Backend and Angular Frontend.

Prerequisites: Ensure you have PHP 8.1+, Composer, Node.js, NPM/Yarn, and MySQL installed.

1. Backend Setup (Laravel 10)

Navigate to the backend directory: domus-naturae-backend-laravel10

1Install Dependencies

composer install

This will install all PHP packages required by the application.

2Environment Configuration

  • Copy the example environment file: 
    cp .env.example .env
  • Open .env and configure your database settings: 
    DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=your_database_name
DB_USERNAME=your_username
DB_PASSWORD=your_password
  • Generate the application key: 
    php artisan key:generate

3Database Migration & Seeding 

php artisan migrate
php artisan db:seed

4Run the Server

php artisan serve

The backend will be available at: http://127.0.0.1:8000

2. Frontend Setup (Angular)

Navigate to the frontend directory: domus-naturae-frontend

1Install Dependencies

npm install
# OR
yarn install

2Run the Server

ng serve

The frontend will be available at: http://localhost:4200

3. Connecting Frontend to Backend

Ensure the frontend is pointing to the correct backend URL. Check src/environments/environment.ts to verify the API URL matches your Laravel server (usually http://127.0.0.1:8000/api).

Troubleshooting

Problem: "Install or enable PHP's gd extension"

If you see an error about ext-gd missing:

  1. Run php --ini to find your php.ini file.
  2. Open it and find the line ;extension=gd.
  3. Remove the semicolon ; to uncomment it: extension=gd.
  4. Save and retry.
Problem: Missing Extensions (zip)

If you see an error about ext-zip missing:

  1. Open php.ini.
  2. Uncomment extension=zip (remove the leading ;).
  3. Save and retry.
Problem: Version Mismatch / Lock File Error

If composer install fails due to PHP version conflicts:

Do not use composer install. Instead, run: 

composer update -W

This regenerates the lock file for your current PHP version.
